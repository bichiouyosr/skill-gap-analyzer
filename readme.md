## 📖 Overview
The **Skill Gap Analyzer** is a tool designed to bridge the gap between a candidate's current skill set and the requirements of their target job role. By analyzing resumes and job descriptions, it highlights missing skills, provides a matched percentage, and suggests actionable learning paths to help candidates reach their career goals.

## ✨ Key Features
* **Resume Parsing:** Extracts current skills, education, and experience from user-uploaded resumes (PDF/DOCX).
* **Job Description Analysis:** Identifies core requirements, keywords, and nice-to-have skills from job postings.
* **Gap Identification:** Compares the two data sets to highlight exactly which skills the candidate is missing.
* **Visual Insights:** Generates easy-to-read charts and match-percentage scores.
* **Learning Recommendations:** (Optional) Suggests courses or topics to close the identified skill gaps.
